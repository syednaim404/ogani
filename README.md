
Project Setup Instructions
---------------------------

1. Download the project from GitHub and extract the ZIP file.
2. Open the extracted project folder in your preferred code editor.
3. Run composer install to install all required packages.
4. Create a database named ogani.
5. Import the ogani.sql file into the ogani database (Note: the SQL file is inside the project folder).
6. Configure the .env file and set the database name to ogani.
7. Run php artisan serve to start the project on the local server.
8. Visit the URL shown in the terminal to view the project in your browser.
